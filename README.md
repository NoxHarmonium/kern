# Kern

Kern is a library of parser combinators for Clojure.

## Installation

Leiningen:

```clojure
[org.blancas/kern "0.1.0"]
```

Maven:

```xml
<dependency>
  <groupId>org.blancas</groupId>
  <artifactId>kern</artifactId>
  <version>0.1.0</version>
</dependency>
```
